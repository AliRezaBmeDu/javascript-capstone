# ğŸ“– Binge-It <a name="about-project"></a>

**Binge-It** aims to create a tv series application. There are multiple shows. You can check their details, like them, comment on those, see the cast and crews(under development). We build the project mostly using javascript with html for layout and basic CSS for design. The project is served by webpack. We used TVMaze API to collect the data and Involvement API to store the like and comments information.
